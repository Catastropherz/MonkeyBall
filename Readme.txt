Controls

Fixed Joystick - Input is controlled by the touch position relative to the centre of the circle.
Dynamic Joystick - As the touch moves further away from its initial touch point the input data is updated.
Gyroscope - This is based on the angle of the device at runtime.

Goal

Reach the Goal Block in each area.

Known issues

Dynamic Joystick doesn't change the position after the first touch. On subsequent touches the joystick remains at the position of the first touch.

However, this is purely a visual bug. The joystick still calculate its direction based on subsequent touches.
